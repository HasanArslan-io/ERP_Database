SQL Server üzerinde geliştirilen örnek ERP veritabanı projesidir.

## Amaç
Müşteri verileri üzerinden temel veri analizi yaparak iş senaryolarına uygun sorgular üretmek.

## İçerik
- Customers tablo yapısı
- Örnek müşteri verileri
- İş senaryolu SQL sorguları

## Kullanılan Konular
- CREATE TABLE
- INSERT INTO
- SELECT
- UPDATE
- DELETE
- GROUP BY
- HAVING

## Ne Yapılıyor?
Bu projede amaç, ham müşteri verisini analiz ederek:
- aktif müşteri takibi
- şehir bazlı dağılım
- müşteri yoğunluğu
gibi iş ihtiyaçlarına yönelik veri üretmektir.
